# 第2大脑

> 记录，成为更好的自己。
> 
> 引用印象笔记推广理念的一句话，“在未来，知道如何找到信息远比记住信息重要”。


写在开始的几句话
1. 建立自己的知识体系，提高自己的认知，作为自己大脑的“外接硬盘”，第2大脑
2. 收集，梳理和沉淀，对自己所需，感兴趣，必备的知识进行整理，并进行“反刍”和思考，提升思想高度
3. “源头活水”，不断吸取新知识，才能自我革新
